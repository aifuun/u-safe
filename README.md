# U-Safe (万能保险箱)

一款专为 U 盘用户设计的轻量化管理工具，集成军事级文件加密与非侵入式标签整理功能。

## 特性

- 🔒 **军事级加密**: AES-256-GCM 认证加密
- 🏷️ **虚拟标签**: 不改变物理路径的智能整理
- 🚀 **无需提权**: 完全用户态运行，无需管理员权限
- 🔄 **跨平台**: 支持 Windows 10/11 和 macOS (Intel/Apple Silicon)
- ⚡ **原生体验**: 基于 Tauri + Rust，性能卓越

## 技术栈

- **后端**: Rust
- **前端**: React + TailwindCSS
- **框架**: Tauri
- **数据库**: SQLite
